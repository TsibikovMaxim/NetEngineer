Запрет на передачу данных для MAC-адресов

Если операционная система IOS обнаружила третий MAC-адрес на порту устройства, помимо разрешённых, она руководствуется настройкам, которые укажет сетевой инженер. В данном случае есть два варианта развития события: либо полностью отключать порт shutdown, либо накладывать ограничение на передачу данных с этого MAC-адреса restrict

Самый оптимальный способ это конечно же наложение ограничений на передачу данных, так как при отключении порта — отключается и POE (Power over Ethernet), то есть вы можете погасить телефон абонента и подключение по локальной сети у компьютера

Switch>en // Привилегированный режим EXEC

Switch#conf t // Режим глобальной конфигурации

Switch(config)#int fa 0/1 // Выбор интерфейса

Switch(config-if)#switchport mode access // Сообщаем системе, что это абонентский порт

Switch(config-if)#switchport port-security // Активируем функцию switchport port-security на интерфейсе fa 0/1

Switch(config-if)switchport port-security violation restrict // Запрещаем передачу с других MAC адресов, которые подключились на порт (выключаем порт командой shutdown)
