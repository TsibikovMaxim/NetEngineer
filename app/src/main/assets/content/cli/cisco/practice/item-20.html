Перезагрузка устройства

Иногда возникает необходимость перезагрузить устройство (маршрутизатор, коммутатор и т.д), сделать это позволяет команда reload.

Switch>en // Привилегированный режим EXEC
Switch#conf t // Режим глобальной конфигурации
Enter configuration commands, one per line. End with CNTL/Z.
Switch(config)#do reload // Команда на перезагрузку
Proceed with reload? [confirm] // Подтверждаем перезагрузку нажатием Enter
