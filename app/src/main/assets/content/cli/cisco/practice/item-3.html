Время старения MAC-адреса

Время старения MAC-адреса очень полезный параметр, который позволяет установить определённое время в течение которого оборудование будет хранить MAC-адрес устройства в своей таблице с MAC-адресами.

По истечении установленного времени, MAC-адрес устройства будет удалён из таблицы.

Switch>en // Привилегированный режим EXEC

Switch#conf t // Режим глобальной конфигурации

Switch(config)#int fa 0/1 // Выбор интерфейса

Switch(config-if)#switchport mode access // Сообщаем системе, что это абонентский порт

Switch(config-if)#switchport port-security // Активируем функцию switchport port-security на интерфейсе fa 0/1

Switch(config-if)#switchport port-security aging time 10 // Устанавливаем время старения MAC-адреса равное 10 минутам
