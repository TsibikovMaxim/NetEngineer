Настройка параметров IPv4 на интерфейсе маршрутизатора

Сейчас мы настроим параметры IPv4 на маршрутизаторе и обеспечим доступность интерфейса.

R1>en // Переходим в привилегированный режим EXEC
R1#conf t // Переходим в режим глобальной конфигурации
Enter configuration commands, one per line. End with CNTL/Z.
R1(config)#interface Serial1/0 // Выбираем интерфейс Serial1/0 для дальнейшей настройки
R1(config-if)#ip address 10.1.1.1 255.255.255.0 // Присваиваем ip-адрес и маску сети
R1(config-if)#description LINK to R2 // Устанавливаем описание для интерфейса
R1(config-if)#no shutdown // Включаем интерфейс
R1(config-if)#
