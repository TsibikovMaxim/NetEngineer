Проверка статических маршрутов IPv6

Наряду с такими командами как ping и traceroute для проверки статических маршрутов также используют несколько следующих команд, которые мы сейчас рассмотрим.

При выполнении этих команд будет выведена информация о всех статических маршрутах, которые присутствуют в таблице маршрутизации.

  • show ipv6 route
  • show ipv6 route static
  • show ipv6 route [ip-адрес]
