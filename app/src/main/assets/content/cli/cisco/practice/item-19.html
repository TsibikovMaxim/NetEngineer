Команда для возвращения назад по дереву конфигурирования

Команда exit или ex позволяет возвращаться назад по дереву конфигурирования, то есть возвращаться к предыдущему этапу при настройке устройства.

Switch>en // Привилегированный режим EXEC
Switch#conf t // Режим глобальной конфигурации
Enter configuration commands, one per line. End with CNTL/Z.
Switch(config)#int vlan 1 // Выбор интерфейса VLAN 1
Switch(config-if)#ex // Возвращаемся в режим глобальной конфигурации
Switch(config)#ex // Возвращаемся в привилегированный режим EXEC
Switch#
%SYS-5-CONFIG_I: Configured from console by console
Switch#ex // Выходим из привилегированного режима EXEC
