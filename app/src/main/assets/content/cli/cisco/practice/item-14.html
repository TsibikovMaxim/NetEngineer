Выключение VLAN

Для отключения нужного VLAN, нужно воспользоваться командой shutdown при конфигурировании интерфейса.

Switch>en // Привилегированный режим EXEC
Switch#conf t // Режим глобальной конфигурации
Enter configuration commands, one per line. End with CNTL/Z.
Switch(config)#int vlan 2 // Выбираем VLAN под номером 2
Switch(config-if)#shutdown // Выключаем VLAN
Switch(config-if)#
%LINK-5-CHANGED: Interface Vlan2, changed state to administratively down
