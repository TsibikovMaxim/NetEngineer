Сохранение файла загрузочной конфигурации во флеш-память

Чтобы сохранить файл загрузочной конфигурации во флеш-память необходимо выполнить команду copy startup-config flash в CLI.

R1# copy startup-config flash // сохраняем файл загрузочной конфигурации во флеш-память
Destination filename [startup-config]? // нажимаем Enter
