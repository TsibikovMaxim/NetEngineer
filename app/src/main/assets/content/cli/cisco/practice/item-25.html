Увеличение времени таймаута exec-timeout

Команда exec-timeout позволяет увеличить время таймаута, то есть увеличить время работы и не отключаться от маршрутизатора или коммутатора.

Запомните: если применить данную команду в такой вариации exec-timeout 0 на консольных линиях и VTY, то система IOS интерпретирует эту команду, как никогда не истекающий таймаут.

Важно: никогда не устанавливайте слишком большое время таймаута из-за соображений информационной безопасности.

Router>en // Переходим в привилегированный режим EXEC
Router#conf t // Переходим в режим глобальной конфигурации
Enter configuration commands, one per line. End with CNTL/Z.
Router(config)#line console 0 // Переходим в настройку консольной линии
Router(config-line)#exec-timeout 60 // Устанавливаем время таймаута в минутах
