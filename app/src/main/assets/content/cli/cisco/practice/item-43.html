Настройка IP-адреса для интерфейса VLAN коммутатора

Чтобы настроить IP-адрес для интерфейса VLAN коммутатора необходимо выполнить следующие команды interface vlan 1 и ip address в CLI.

S1# conf t // переходим в режим глобальной конфигурации
S1(config)# interface vlan 1 // переходим в режим конфигурирования интерфейса
S1(config-if)# ip address 192.168.1.253 255.255.255.0 // присваиваем интерфейсу ip-адрес и маску сети
S1(config-if)# no shutdown // включаем интерфейс
S1(config-if)# ex // выходим из режима глобальной конфигурации
S1#
