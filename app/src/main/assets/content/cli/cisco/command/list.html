Команды Cisco

На главную

  • show version

    Информация об обслуживании

  • show ip interfaces brief

    Информация об портах и vlan

  • sh interfaces status

    Информация о интерфейсах

  • sh ip protocols

    Информация о настроенных протоколах

  • sh vlan

    Информация о vlan

  • sh vlan brief

    Подробная информация о vlan

  • sh users

    Информация о подключенных пользователях

  • sh ntp status

    Сведения о состоянии синхронизации сетевого времени

  • sh ssh

    Проверка SSH-подключений к устройству

  • sh ip ssh

    Информация о ssh

  • sh mac address-table static

    Информация о статически прописанных разрешенных адресах

  • sh port-security address

    На каком порту какой адрес настроен

  • sh interfaces fastethernet0/1 switchport

    Статус всех коммутационных портов или указанного порта

  • sh run | begin <word>

    Показать все начиная с word

  • sh run | include <word>

    Показать все строки с word

  • terminal monitor

    Включение отображения monitor logging
